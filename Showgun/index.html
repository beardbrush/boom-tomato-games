Samurai Card Games

Same rice-paper deck. Five games. Minimal luck, lots of decisions.

Choose a Game

1) Iai Duel (Secret Pick)
Hand tactics + timing. Pick a card secretly; reveal; score points. Low luck, high mind-game.
FastSkill: ★★★
2) Ronin Tricks (Trick-Taking Lite)
Follow clan if you can. Trump clan rotates each match. Count what’s been played.
ClassicSkill: ★★★
3) Dojo Relics (Market + Sets)
Draft from a market, build sets, claim points. Planning + denial; looks gorgeous.
ChillSkill: ★★☆
4) Shogun Showdown (Poker-lite)
Draw 5, keep 3. Score 3-card hands (pair/straight/flush/trips). Quick dopamine.
SnappySkill: ★★☆
5) Skirmish Lanes (3 Lanes Battle)
Place cards into 3 lanes. Combos + lane control. First to win 2 lanes wins.
TacticalSkill: ★★★

All games are offline and asset-free (cards are generated SVGs). If you want next, we can add: sound, clan techniques, campaign opponents, and “unlockable” card backs.
